# Звуковые файлы

## alarm.mp3

Добавьте сюда файл `alarm.mp3` - короткий тревожный звук (бип, сирена, звонок).
Рекомендации:
- Длительность: 1-3 секунды
- Формат: MP3
- Громкий и заметный звук

Если файл отсутствует, приложение автоматически использует системные уведомления + вибрацию как запасной вариант.
